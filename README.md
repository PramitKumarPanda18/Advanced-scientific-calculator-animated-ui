# Advanced Scientific Calculator (Animated UI)

An interactive **advanced scientific calculator web application** built using **HTML, CSS, and JavaScript**.
This project features a **modern animated UI**, **floating math particle effects**, **scientific functions**, and **calculation history tracking**.

---

## 🚀 Features

* Basic arithmetic operations (`+`, `−`, `×`, `÷`)
* Scientific functions (`sin`, `cos`, `tan`, `log`, `√`)
* Mathematical constants (`π`, `e`)
* Power calculation (`^`)
* **DEG / RAD mode toggle** for trigonometric calculations
* Calculation **history panel**
* **Clear history** functionality
* **Backspace support**
* Floating **math particle background animation**
* **Animated calculator UI**
* Result flash animation when calculations are performed
* Clean and responsive layout
